MetaGenScope-CLI is used to upload data sets to the MetaGenScope web platform.

## Getting Started

### Installation

If you don't use `pipsi`, you're missing out.
Here are [installation instructions](https://github.com/mitsuhiko/pipsi#readme).

Simply run:

    $ pipsi install .


### Usage

To use it:

    $ metagenscope --help
